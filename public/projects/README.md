# Project Assets

This directory stores project screenshot images used across the portfolio.

## Image Naming Convention

Each project folder must contain the following image files:

| File            | Description              |
|-----------------|--------------------------|
| `hero.webp`     | Hero / banner screenshot |
| `full.webp`     | Full page screenshot     |
| `homepage.webp` | Homepage screenshot      |
| `collection.webp` | Collection page        |
| `product.webp`  | Product page screenshot  |
| `cart.webp`     | Cart page screenshot     |
| `mobile.webp`   | Mobile viewport         |
| `feature-1.webp` | Feature highlight 1    |
| `feature-2.webp` | Feature highlight 2    |
| `feature-3.webp` | Feature highlight 3 (optional) |

## Desktop Screenshots

| Property  | Value    |
|-----------|----------|
| Width     | 1440px   |
| Zoom      | 100%     |
| Format    | WebP     |
| Quality   | High     |

## Mobile Screenshots

| Property  | Value      |
|-----------|------------|
| Viewport  | 390 × 844  |
| Format    | WebP       |

## Naming Rules

- Use lowercase.
- Use hyphens only.
- No spaces.
- No special characters.
